# CodeAlpha_HangmanGame
  
  A terminal based implementation of Hangman- a classic word guessing game using Python.
  Player aka user attempts to uncover a randomly selected word from the pre-defined dictionary in the code.
  With each guess, live feedback and remaning attempts are displayed.
  Features replay functionality and clean game state display.
  This is built as a part of CodeAlpha Summer Intership Program in Python programming.
